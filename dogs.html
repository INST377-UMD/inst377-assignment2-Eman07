Audio Navigation Instructions

You can use the following voice commands to navigate this website:

  • "Hello" - Alerts with Hello World
  • "Change the color to [color]" - Changes the background color of the page
  • "Navigate to [page]" - Navigates to the specified page (Home, Stocks, or Dogs)
  • "Load Dog Breed [breed name]" - Displays information for the specified dog breed
